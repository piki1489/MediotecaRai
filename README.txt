# MediotecaRai

MediotecaRai es una aplicación web desarrollada con ASP.NET Core MVC.  
El proyecto sirve para gestionar y mostrar contenidos multimedia de forma organizada.

## Tecnologías utilizadas

- ASP.NET Core MVC
- .NET
- Razor
- Entity Framework Core
- SQL Server
- HTML, CSS y JavaScript

## Estructura del proyecto

- Controllers  
  Contiene los controladores de la aplicación.

- Models  
  Contiene los modelos y ViewModels.

- Views  
  Contiene las vistas Razor.

- Data  
  Contiene el contexto de base de datos y la lógica de acceso a datos.

- wwwroot  
  Contiene los archivos estáticos (css, js, imágenes).

## Configuración del proyecto

1. Clonar el repositorio.
2. Abrir la solución en Visual Studio.
3. Configurar la cadena de conexión en el archivo `appsettings.json`.
4. Ejecutar el proyecto.

## Ejecución

El proyecto se puede ejecutar desde Visual Studio usando IIS Express o el servidor integrado.
